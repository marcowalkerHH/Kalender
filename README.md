## Main-Verzeichnis aktualisieren

Führe `./sync_main.sh` aus, um das Main-Verzeichnis mit den aktuellen Dateien (`index.html`, `style.css`, `script.js`) zu aktualisieren. Das Skript kopiert die Dateien automatisch in den Ordner `Main`, sodass sie anschließend nach Gitta hochgeladen werden können.
